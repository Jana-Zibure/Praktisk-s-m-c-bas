Python, kodu jūrā lēnām peld

Python, kodu jūrā lēnām peld,
Ar vienkāršību, kas prātus sedz.
No pirmās rindiņas līdz pēdējai,
Vieglums un skaidrība neiztrūkst tai.

Bez liekiem punktiem, komatiem rindās,
Kods plūst, kā upe, mierīgs savās straumēs.
Ar mainīgajiem, funkcijām spēlējas,
Cik saprotami un vienkārši tie atklājas.

If un else, tie ceļu norāda,
While un for, ciklu ritmu glabā.
Ar lambda un map var variēt,
Un list comprehension jaunu ceļu rādīt.

Datu struktūras Pythonā plašas,
Listes, vārdnīcas – tās visur der.
Ar Pandas, NumPy, datu analīze,
Machine learning ar scikit-learn gaisā nes.

Java kods kā mūzika plūst

Java kods kā mūzika plūst,
Kas rindiņu pēc rindiņas rūpīgi lūst.
Tā valoda, kas strādā, kur vien mēs būsim,
Pasauli tehnoloģijās kopā mēs būsim.

Ar kompīleriem ceļu mēs vedam,
Objektos, kas datus mums dod, mēs ejam.
Izpildes laikā, kad kods skanēs,
Dzīvības elpu mašīnās mēs celsim.

Platformu neatkarību Java svētī,
"Vienreiz uzraksti – visur izpildi" – dziesma tā vēsti.
No viedtālruņiem līdz serveriem lieliem,
Visur Java zvaigznes debesis mums piepilda.

Ar bibliotēkām plašām un jaudīgu rīku,
Java valoda kļūst par mūsu draugu un dīku.
Kods kā dzeja, kas dzīvē var būt,
Java valoda palīdz mūsu sapņus skūpstīt.

Autors: Aļona Čerica
